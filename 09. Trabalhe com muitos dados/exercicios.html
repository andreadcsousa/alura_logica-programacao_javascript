É possível criar um array a partir das informações de um usuário. Inserindo, inclusive, tipos diferentes de informações: texto, número, "booleano", como no exemplo abaixo.

Para acessar o item de uma lista, deve-se contar a partir do 0 (zero). Então para escolher a pizza de atum, o item correto do array é "pizzas[2]", para acessar esse elemento em específico.

Para saber o tamanho do "array" pode-se utilizar o ".length", uma propriedade que usa quantidade de elementos da lista como limite da condição.

Para adicionar outro item na lista, basta utilizar a propriedade ".push", ela serve para inserir elementos no "array" sem precisar editar a lista manualmente.
